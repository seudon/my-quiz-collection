# My Quiz Collection

オフラインで動作する学習用クイズアプリケーション集です。HTML/CSS/JavaScriptのみで構築されており、ブラウザで直接開くだけで使用できます。

## 特徴

- 📴 **完全オフライン動作** - インターネット接続不要
- 🎯 **多彩な出題形式** - 4択問題と入力形式の両方に対応
- 📱 **レスポンシブデザイン** - スマートフォンでも快適に利用可能
- ⌨️ **キーボードショートカット** - Enter（次へ）、1-4キー（選択）
- 📊 **詳細な結果表示** - 正解率、間違えた問題の一覧、全問題レビュー
- 🔄 **間違い復習機能** - 間違えた問題だけを再度学習可能

## 収録クイズ

### 🔬 科学
- **地質時代クイズ** - カンブリア紀〜白亜紀の期間・開始・終了・境界（11データ）

### 🔢 数学・日常計算
- **分数⇄小数⇄百分率クイズ** - 1/2=0.5=50%など日常でよく使う分数を即答（31データ）
- **2の累乗クイズ** - 2^10=1024など、コンピュータで頻出する2の累乗を暗記（23データ）

### 🌍 地理
- 近日公開予定

## 使い方

### 基本的な使い方

1. ブラウザで `index.html` を開く
2. 学習したいクイズを選択
3. 設定（モード・出題数・範囲）を選択
4. 「クイズ開始」ボタンをクリック
5. 問題に回答して「次へ」で進む
6. 最後に結果を確認

### オフライン利用方法

1. このリポジトリをダウンロードまたはクローン
2. ローカルに保存
3. `index.html` をブラウザで開く

それだけです！サーバーやビルドツールは不要です。

### キーボードショートカット

- **Enter** - 次の問題へ進む
- **1-4** - 選択肢を選択（4択モード時）

## 新しいクイズの作成方法

### 1. テンプレートをコピー

既存のクイズ（例：`quizzes/science/geology.html`）をコピーして、適切なカテゴリフォルダに配置します。

### 2. クイズデータを定義

HTML内のJavaScript部分でクイズデータを定義します：

```javascript
const myQuizData = [
  {
    id: "item1",
    name: "項目名",
    // クイズ固有のデータフィールド
    kind: "category",
    explain: "説明文"
  },
  // ...
];
```

### 3. QuizEngineを継承

クイズ固有のロジックを実装します：

```javascript
class MyQuizEngine extends QuizEngine {
  constructor() {
    super(myQuizData, {
      title: 'クイズタイトル',
      description: 'クイズの説明'
    });
  }

  makeQuestion(item, mode, focus) {
    // 問題生成ロジック
    return {
      id: item.id + "-" + Math.random().toString(36).slice(2),
      item,
      prompt: "問題文",
      correct: "正解",
      choices: ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
      type: "question-type",
      mode,
      badges: ["バッジ1", "バッジ2"],
      hint: "ヒント",
      placeholder: "入力例"
    };
  }

  compareAnswer(userAnswer, correctAnswer) {
    // 回答比較ロジック（必要に応じてオーバーライド）
    return userAnswer === correctAnswer;
  }
}
```

### 4. インデックスページに追加

`index.html` のカテゴリセクションにクイズカードを追加します。

## カスタマイズ

### 色テーマの変更

`common/quiz-common.css` の `:root` セクションでカラー変数を変更できます：

```css
:root {
  --bg: #0f172a;
  --accent: #22c55e;
  --accent2: #06b6d4;
  /* ... */
}
```

### 設定項目の追加

各クイズHTMLの設定セクションに新しい `<select>` や `<input>` を追加し、JavaScript側で対応します。
